üöÄ Getting Started

Welcome to Head Copy! This app provides realistic Morse code training with multiple specialized modes designed for amateur radio operators.

Quick Start

  1. Set up your Station Information (CallSign, Name, QTH) in the Station tab
  2. Choose your training mode in Settings
  3. Adjust WPM Speed (5-50 WPM, Premium unlocks 20+ WPM)
  4. Set your practice Duration (1-60 minutes, Premium unlocks 5+ minutes)
  5. Configure audio Frequency (400-1000 Hz)
  6. Start practicing in the Practice tab
üí° Head Copy Technique: Enable "Display Delay" to see text AFTER hearing the Morse code - this trains true head copy skills where you decode in your mind before seeing the answer.

‚öôÔ∏è Settings & Configuration

Morse Code Settings

Speed Controls

  • WPM Speed (5-50): Overall transmission speed
  • Farnsworth Speed: Character spacing (cannot exceed WPM)
  • Wordsworth Speed: Word spacing timing (cannot exceed WPM)

Timing & Audio

  • Duration (1-60 minutes): Practice session length
  • Frequency (400-1000 Hz): Morse tone frequency
  • Display Delay: Show text before (off) or after (on) morse plays

Premium Features

  • High Speed: Unlock WPM speeds above 20
  • Extended Sessions: Practice for more than 5 minutes
  • Radio Sound Settings: Advanced audio processing
‚ö†Ô∏è Head Copy Training: Enable "Display Delay" for true head copy practice - you'll hear the Morse first, then see the text. This builds real-world copying skills.

üèóÔ∏è Station Setup

Configure your station information for realistic QSO simulations.

Required Information

  • Your CallSign: Your amateur radio call sign
  • Name: Your first name or nickname
  • QTH (SPC Code): Your state/province/country

Club Memberships (Optional)

  • CWops Number: For CWops CWT contests
  • SKCC Number: For SKCC operations (supports S, T, C suffixes)
  • LICW Number: For LICW club activities

Personalization

  • Custom Message: Replace the default "May The Morse Be With You" with your own motivational message

üéØ Practice Interface

Main Practice Screen

Display Elements

  • Your CallSign: Prominently displayed when not practicing
  • Motivational Message: Custom or default inspiration
  • UTC Clock: Current UTC date and time
  • Local Clock: Your local date and time
  • Training Mode Info: Current mode, speed, duration, and frequency

Practice Session

  1. Tap the play button to enter full-screen practice mode
  2. Watch the 3-2-1 countdown
  3. Listen to Morse code transmissions
  4. Text appears based on your Display Delay setting
  5. Use media controls (headphone buttons) for pause/resume/stop
  6. Session ends automatically after set duration

Full-Screen Mode

  • Immersive Practice: Large text display on black background
  • Exit Control: X button with confirmation dialog
  • Media Controls: Play/pause button in corner
  • Countdown Timer: Remaining session time
  • Prosign Display: + becomes <AR>, = becomes <BT>, ( becomes <KN>

üéØ Practice Tip: Regular short sessions are more effective than occasional long ones. The app tracks sessions as short as 30 seconds to encourage consistent daily practice.

üí° Practice Tips

Head Copy Technique

  • Enable Display Delay: See text AFTER hearing code
  • Don't Rush: Let the complete thought form before looking
  • Practice Daily: 15-20 minutes daily beats long weekly sessions
  • Realistic Speeds: Practice at speeds you'll use on-air

Mode-Specific Tips

Contest Practice

  • Focus on exchange formats for your target contests
  • Practice with station information that matches contest requirements
  • Use realistic serial number ranges

POTA Practice

  • Learn common park references and callsign patterns
  • Practice both activator and hunter perspectives
  • Focus on efficient exchanges

Koch Method

  • Add new characters only when achieving 90% accuracy
  • Maintain character speed at 15+ WPM
  • Focus on character sounds, not visual patterns

üîß Troubleshooting

Common Issues

No Audio

  • Check device volume and silent mode
  • Verify app isn't muted in Control Center
  • Test with headphones to isolate speaker issues
  • Restart the app

Premium Features Not Working

  • Check subscription status in Settings > Subscriptions
  • Try "Restore Purchases" in app settings
  • Sign out and back into Apple ID
  • Restart device

Station Information Required Alerts

  • Complete your CallSign, Name, and QTH in Station tab
  • Add relevant club numbers for contest modes
  • Some modes require specific information to function
